# Curso Técnico em Redes de Computadores

Este repositório contém o material didático para o curso técnico em Redes de Computadores, ministrado pelo Professor Maxwell de Oliveira Chaves.

## 📚 Disciplinas

O curso é dividido em duas principais disciplinas:

### 1. Infraestrutura Física de Redes
- Estudo dos componentes físicos das redes de computadores
- Cabos, conectores, topologias e protocolos de rede

### 2. Oficina de Montagem e Conexão
- Montagem e manutenção de computadores
- Componentes de hardware, alimentação elétrica e conexões

## 🌐 Acesso às Aulas

Para acessar o material didático:

1. Abra o arquivo `index.html` no seu navegador
2. Selecione a disciplina desejada
3. Navegue pelas aulas disponíveis

Alternativamente, acesse diretamente as pastas:
- **Infraestrutura Física:** `web/ifr/aulas.html`
- **Oficina de Montagem:** `web/omc/aulas.html`
